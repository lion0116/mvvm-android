貓印 用戶隱私政策

本《用戶隱私政策》（下稱《政策》）說明了 貓印 平臺（下稱“貓印”）在收集、管理和保護用戶個人資訊方面的政策及措施。
您的注册、登入、使用等行為將視為您已知悉並同意 貓印 平臺按《政策》收集、管理和保護您的個人資訊。
貓印 有權隨時修改《政策》的任何條款，用戶可在 貓印 網站和移動應用中查閱最新版《政策》的條款。如果用戶不同意 貓印 對《政策》相關條款所做的修改，用戶有權立即停止使用 貓印 服務。用戶繼續使用 貓印 服務將被視為接受 貓印 對本《政策》相關條款所做的修改。

一、個人資訊的收集

在您注册、登入、使用 貓印 服務時，我們可能會收集您提供的相關個人資訊及其他非個人隱私資訊，包括但不限於：手機號、IP地址、地理位置、IMEI號、MAC地址、設備型號、操作日誌以及通過cookie和web beacon採集到的科技資訊等。
貓印 承諾採集cookie和web beacon資訊僅用於記住用戶的登入狀態、判斷用戶使用 貓印 服務的情况，目的是提升服務質量及優化用戶體驗。

二、個人資訊的管理

貓印 不對外公開或向協力廠商提供用戶的注册資料及用戶在使用 貓印 服務時存儲在本平臺的非公開內容，但下列情况除外：

  1. 事先獲得用戶或其監護人的明確授權；
  2. 按照司法機關或行政機關的法定程式要求；
  3. 貓印 為維護自身合法權益而向用戶提起訴訟或仲裁時；
  4. 有關法律法規規定的其他情形。

三、個人資訊的保護

貓印 注重保護其獲得的用戶個人資訊。為防止用戶個人資訊在意外的、未經授權的情况下被非法訪問、複製、修改、傳送、遺失、破壞、處理或使用，貓印 已經並將繼續採取以下措施保護您的個人資訊：

  1. 以適當的管道對用戶的個人資訊進行加密處理；
  2. 在適當的位置使用密碼對用戶個人資訊進行保護；
  3. 限制對用戶個人資訊的未授權訪問；
  4. 其他合理措施。

此外，用戶應採取積極措施保證個人資訊的安全，比如：定期修改帳號密碼、不要將自己的帳號密碼等個人資訊透露給他人。

四、未成年人保護

貓印 非常重視對未成年人個人資訊的保護。若您是18周歲以下的未成年人，在使用 貓印 服務前，應確保已取得監護人的同意，如您在 貓印 上申請註冊帳號，貓印 將視為您已得到前述同意。貓印 將根據國家相關法律法規及本《政策》的規定保護未成年人的個人資訊。
