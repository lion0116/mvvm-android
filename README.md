# mvvm-android

项目简介
--------------------

本项目本意在于探究Android项目的MVVM架构设计，较好的利用Google提供的ViewModel、LiveData、DataBinding等组件，
结合Retrofit、Fresco等网络、图片框架，快捷地搭建起一个结构清晰、遵循设计模式原则的、便于阅读维护的项目。
